# Datawire

Visual data pipelines in the browser: build a graph, configure nodes, run it (Web Worker), and share via invite links.

## Getting Started

### 1) Configure env vars

Copy:

```bash
cp .env.example .env.local
```

Fill in `.env.local` using your Supabase project + OAuth credentials.

### 2) Run the dev server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
